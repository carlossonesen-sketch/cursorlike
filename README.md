# DevAssistantCursorLite

A local-first AI coding assistant inspired by tools like Cursor, designed to analyze code, suggest edits, and assist with development workflows.

## Key Features
- Project-aware file reading and navigation
- AI-powered code suggestions and edits
- Local model integration (llama / local runtime)
- OpenAI API support
- Multi-file editing workflow

## Tech Stack
- TypeScript
- React
- Tauri
- Node.js

## What I Built
Designed and implemented the architecture for a local AI coding assistant, including intent routing, file handling, and model integration.
